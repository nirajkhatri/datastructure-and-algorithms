## Headings in Markdown

---

- ### In Markdown there are Six levels of Heading ranging from 1 to 6. Heading size decreases as we go on ascending order
- ### Headings In MarkDown :
  ```javascript
  # -> Determine Heading Size
  # H1
  ## H2
  ### H3
  #### H4
  ##### H5
  ###### H6
  ```
